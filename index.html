Folio.js

jQuery-плагин для презентаций

Установка и настройка

Для активации слайдов нужно подключить скрипт, заполнить элемент слайдами и вызвать метод folio() для него.

Готово!

А когда все готово, можно посмотреть картинки с котиками :)

Вертикальные слайды

Для слайда картинок сверху вниз нужно добавить класс folio-vertical к контейнеру
